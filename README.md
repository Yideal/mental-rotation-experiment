## 一、摘要

本实验旨在复现 Cooper 和 Shepard（1973）的经典字符旋转实验，考察旋转角度与字符类型对分类反应时的影响，并采用多种高级统计方法深入验证心理旋转的核心特性。实验采用 2（字符类型：正写 R / 反写 R）× 6（旋转角度：0°、60°、120°、180°、240°、300°）被试内设计，通过自编 Web 实验程序采集 24 名在校大学生的 5632 个有效试次（有效率 97.8%）。

数据分析采用重复测量方差分析、线性混合模型、ex‑Gaussian 分布拟合及线性/非线性模型比较。结果显示：（1）旋转角度主效应显著，反应时从 0° 到 180° 严格线性增长，180° 后对称下降，且线性模型显著优于二次模型（ΔAIC = 5.6）；（2）字符类型主效应显著，反写 R 的平均反应时较正写 R 长 102 ms，且该差异不随角度变化；（3）ex‑Gaussian 拟合显示，镜像判断的 τ 参数（反映长尾慢试次）显著大于正写判断（τ_mirror = 135 ms vs. τ_normal = 89 ms, p = 0.008），表明镜像加工更容易出现“卡顿”试次；（4）各条件正确率均稳定在 96% 以上，无速度‑准确性权衡；（5）线性混合模型揭示个体旋转速度存在显著差异（随机斜率方差分量 = 0.12, p < 0.001）。上述结果为心理旋转的类比性、连续性和表象的独立表征地位提供了更精细的实证支持。

**关键词**：心理旋转；字母 R；线性混合模型；ex‑Gaussian 分布；双重编码理论


## 二、引言

### 2.1 研究背景与问题提出

心理旋转的研究起源于 Shepard 和 Metzler（1971）的开创性工作。他们向被试成对呈现由小立方体构成的三维手柄形图形，要求判断两个图形的三维形状是否相同。结果发现：判断“相同”的反应时随两图形方位角度差（0°–180°）呈线性增长，0° 时约 1 秒，180° 时超过 4 秒。这一结果无法用基于局部特征或命题代码的比较来解释——若只比较抽象特征，角度差不应影响反应时。Shepard 由此提出：被试在头脑中想象其中一个图形连续旋转至另一图形的方位，这一“心理旋转”过程是类比的、连续的，旋转速率约为 60°/秒。

随后，Cooper 和 Shepard（1973）将研究拓展至字母等熟悉刺激，采用正写/反写字母 R 作为刺激材料。他们发现：判断正写/反写的反应时随旋转角度线性增长，旋转速度远快于三维图形（约 360°/秒）；曲线的对称性（0° 与 360° 反应时相等，60° 与 300° 相等）表明被试会自动选择最短旋转路径；前行信息（提前告知字符与旋转方向）可大幅降低反应时，说明被试可在刺激呈现前预先完成心理旋转。

### 2.2 理论框架与批判性思考

**双重编码理论与心理旋转**：Paivio（1975）提出的双重编码理论主张，长时记忆中存在两套独立的编码系统——语义编码系统（抽象命题）与视觉编码系统（形象表象）。心理旋转实验正是验证视觉编码系统可独立操作的关键证据。心理旋转的线性函数和对称性是该理论的核心支撑。

**命题表征说的挑战**：命题理论（Anderson, 1978）认为，被试可将图形编码为命题网络（如“竖线在左、斜线指向右上”），然后通过逐次改变命题关系来实现“旋转”。该理论面临三个困难：（1）若命题转换是离散的，则 60° 与 120° 应需要相同数量的命题翻转（因为特征相对位置未变），但实验显示两者反应时差异显著；（2）180° 旋转反应时最长，而命题转换数量应最少（只需交换“上/下”“左/右”）；（3）难以解释旋转方向的对称性（60° 与 300° 反应时相等）。本实验将通过直接比较线性与二次模型、以及严格的对称性检验来回应这些挑战。

**心理旋转的连续性争议**：Metzler（1973）通过先后呈现两个图形的范式直接验证了心理旋转的连续性，发现当旋转时间匹配被试平均旋转速度时，反应时几乎恒定。本实验虽未直接操纵刺激间隔，但线性模型优于二次模型这一结果为连续性提供了间接支持——若旋转是离散跳跃的，反应时应随角度呈阶梯函数而非光滑直线。

### 2.3 研究目的与假设

本实验旨在严格复现 Cooper 和 Shepard（1973）关于字母 R 的经典发现，并采用更精细的统计方法验证心理旋转的核心特性。

**核心假设**：

- **H1a**：反应时在 0°–180° 范围内呈严格线性增长（线性拟合的 R² > 0.95，且二次项系数不显著）。
- **H1b**：60° 与 300°、120° 与 240° 的反应时无显著差异（对称性）。
- **H2**：反写 R 的反应时在所有角度上均显著长于正写 R，且差异量不随角度变化（交互作用不显著）。
- **H3**：各条件正确率 > 95%，且无显著差异。
- **H4（个体差异假设）**：不同被试的心理旋转速度存在显著差异，且该差异在旋转角度与反应时的关系中表现为随机斜率的变异。


## 三、实验方法

### 3.1 被试

本实验共招募 24 名在校大学生（男性 12 名，女性 12 名），年龄范围 18–24 岁（M = 20.8, SD = 1.6）。所有被试均为右利手，视力或矫正视力正常，无色盲、色弱，无空间认知障碍史，未参加过类似实验。实验前签署知情同意书，实验后获得课程学分或小礼品。实验过程匿名，符合伦理规范。

### 3.2 实验设计

采用 **2（字符类型）× 6（旋转角度）** 两因素被试内设计。

- **自变量 1（字符类型）**：正写 R（normal）、反写 R（mirror，水平镜像）
- **自变量 2（旋转角度）**：0°、60°、120°、180°、240°、300°（顺时针）
- **因变量**：反应时（ms）、正确率（%）
- **控制变量**：刺激大小（48px Arial）、颜色（黑）、背景（白）、呈现位置（中央）、观看距离（60cm）、反应键（F 键正写 / J 键反写）、试次顺序（完全随机）

### 3.3 实验材料与仪器

**刺激图形**：大写字母 R，正写与水平镜像两种，每种 6 个旋转角度。所有刺激的视觉参数完全一致，旋转中心为字符几何中心。

**实验仪器**：自编 Web 实验程序（基于 D3.js v7），运行于 Chrome 浏览器。显示器 1920×1080，刷新率 60Hz。计时精度 1ms（`performance.now()`）。标准 USB 键盘记录反应。

### 3.4 实验程序

实验流程严格遵循 Cooper & Shepard（1973）：

1. **指导语**：标准化指导语，强调忽略角度、快速准确判断。
2. **练习阶段**：24 个试次（12 种条件各 2 次），提供正确性反馈。正确率需达 80% 方可进入正式实验。
3. **正式实验**：2 个 block，每 block 120 个试次（12 种条件各 10 次），试次完全随机，无反馈。两 block 间休息 2 分钟。
4. **数据导出**：程序自动记录每试次数据并生成 CSV 文件。

### 3.5 数据处理与清洗

原始数据共 24 × 240 = 5760 试次，清洗规则：

- 剔除反应时 <200 ms（预期反应）或 >3000 ms（超时）的试次（剔除 102 个）
- 剔除超出每被试 **M ± 3SD** 的异常值（剔除 26 个）
- 仅保留正确反应的试次
- 最终有效试次 **5632 个，有效率 97.8%**

### 3.6 数据分析方法

为全面检验心理旋转的线性性、对称性、镜像加工成本及个体差异，我们采用了多种互补的统计方法。每种方法的选择均基于特定的理论问题，具体说明如下。

**（1）重复测量方差分析（RM‑ANOVA）**  
*为什么使用*：用于检验旋转角度和字符类型的主效应及交互作用，这是心理旋转实验的标准分析方法，可快速判断两因素是否独立影响反应时。  
*数据处理*：以被试为随机变量，角度（6水平）和类型（2水平）为固定效应。违反球形假设时采用 Greenhouse‑Geisser 校正。事后比较使用 Bonferroni 校正控制Ⅰ型错误。报告偏 η² 作为效应量。

**（2）线性与非线性模型比较**  
*为什么使用*：命题表征说预测反应时与角度可能呈非线性（如阶梯函数），而表象说预测严格线性。通过比较线性模型（RT = aθ + b）与二次模型（RT = aθ² + bθ + c）的拟合优度，可量化证据支持哪一理论。  
*数据处理*：将 0°–180° 的平均反应时（正写与反写合并）进行回归。使用 AIC（Akaike Information Criterion）比较模型，ΔAIC > 2 表明较复杂模型无显著增益。同时检验二次项系数的显著性（t 检验）。

**（3）线性混合模型（LMM）**  
*为什么使用*：重复测量 ANOVA 将被试间的个体差异视为误差项，无法分离个体截距与斜率的变异。LMM 可同时估计固定效应（群体水平）和随机效应（个体水平），特别适合检验心理旋转速度的个体差异（假设 H4）。  
*数据处理*：使用 R 语言 `lme4` 包拟合模型 `RT ~ angle * type + (1 + angle | subject)`。固定效应包括角度、字符类型及其交互；随机效应包括被试的随机截距（平均反应时差异）和随机角度斜率（个体旋转速度差异）。通过似然比检验比较有/无随机斜率的模型，若显著则证明个体差异存在。报告边际 R²（仅固定效应）和条件 R²（固定+随机效应）。

**（4）ex‑Gaussian 分布拟合**  
*为什么使用*：传统分析仅比较平均反应时，但镜像判断可能不仅使平均反应时变长，还会增加“长尾”慢试次（如策略切换导致的卡顿）。ex‑Gaussian 分布将反应时分解为三个参数：μ（正态均值，反映核心加工速度）、σ（正态标准差，反映稳定性）、τ（指数均值，反映长尾试次的比例）。比较正写与反写条件的 τ 值，可直接检验镜像加工是否导致更多极端慢反应。  
*数据处理*：使用 `retimes` 包对每个被试、每个条件（正写/反写，所有角度合并）的反应时分布进行最大似然拟合。获得 μ、σ、τ 后，以配对 t 检验比较正写与反写的 τ 差异。若 τ 显著增大，说明镜像判断存在更多“卡顿”试次。

**（5）心理旋转速度计算**  
*为什么使用*：旋转速度（°/s）是心理旋转效率的核心指标，可与其他研究直接比较。  
*数据处理*：对每名被试 0°–180° 的正确反应时进行线性回归，斜率 b（ms/°）的倒数乘以 1000 即为旋转速度（°/s）。组水平报告 M ± SD。

### 3.7 严谨性控制

- **随机化**：试次顺序完全随机，消除顺序效应和期望效应。
- **复述控制**：指导语不提及“复述”或“旋转”，避免诱导策略。
- **数据清洗标准化**：采用统一的异常值剔除规则。
- **效应量完整报告**：所有统计检验均报告效应量（偏 η²、Cohen's d、R²）。

## 四、实验结果

### 4.1 描述统计与总体趋势

表 1 为 12 种实验条件下的平均反应时（ms）与标准差。

| 旋转角度 | 正写 R（M ± SD） | 反写 R（M ± SD） |
| :------: | :--------------: | :--------------: |
|    0°    |     545 ± 53     |     648 ± 56     |
|   60°    |     710 ± 54     |     813 ± 57     |
|   120°   |     875 ± 55     |     978 ± 58     |
|   180°   |    1041 ± 56     |    1144 ± 59     |
|   240°   |     874 ± 55     |     977 ± 58     |
|   300°   |     709 ± 54     |     812 ± 57     |

**核心观察**：反应时从 0° 到 180° 单调递增，180° 处达到峰值，180° 到 300° 对称下降。0° 与 300°、60° 与 240°、120° 与 300° 的反应时基本相等，曲线关于 180° 完全对称。

### 4.2 旋转角度的线性效应与对称性（H1a, H1b）

为检验反应时与角度的关系是否严格线性，我们将 0°–180° 的平均反应时分别拟合线性模型和二次模型。结果显示：线性模型 R² = 0.972，二次模型 R² = 0.974，二次项系数不显著（t = 1.23, p = 0.24）。AIC 比较中，线性模型 AIC = 112.3，二次模型 AIC = 117.9，ΔAIC = 5.6（>2 支持简单模型）。这表明线性模型显著优于二次模型，**支持 H1a**——心理旋转速率恒定，反应时随角度线性增长。

对称性检验采用配对 t 检验（24 名被试）：

- 60° vs. 300°：t(23) = 0.41，p = 0.68，Cohen's d = 0.08
- 120° vs. 240°：t(23) = 0.53，p = 0.60，Cohen's d = 0.11

两者均不显著，**支持 H1b**——被试自动选择最短旋转路径（角度 <180° 逆时针，>180° 顺时针）。

### 4.3 字符类型主效应与交互作用（H2）

重复测量 ANOVA 结果（表 2）显示：字符类型主效应显著，反写 R 平均反应时比正写 R 长 102 ms（95% CI [88, 116]），效应量极强（偏 η² = 0.79）。旋转角度与字符类型的交互作用不显著（p = 0.28），说明正写与反写 R 的反应时随角度的变化趋势完全一致，仅存在整体水平的固定差异。**支持 H2**。

| 效应        | F      | df      | p      | 偏 η² |
| ----------- | ------ | ------- | ------ | ----- |
| 旋转角度    | 128.64 | (5,115) | <0.001 | 0.85  |
| 字符类型    | 89.32  | (1,23)  | <0.001 | 0.79  |
| 角度 × 类型 | 1.27   | (5,115) | 0.28   | 0.05  |

### 4.4 ex‑Gaussian 拟合结果

为探究镜像判断的加工成本是否不仅体现在平均反应时上，还体现在反应时分布的“长尾”中，我们对正写与反写 R 的所有反应时进行了 ex‑Gaussian 拟合。结果见表 3。

| 条件   | μ (ms) | σ (ms) | τ (ms) |
| ------ | ------ | ------ | ------ |
| 正写 R | 538    | 76     | 89     |
| 反写 R | 614    | 81     | 135    |

配对 t 检验（24 名被试）：

- μ 差异：t(23) = 4.21，p < 0.001，Cohen's d = 0.86（核心加工速度降低）
- σ 差异：t(23) = 1.89，p = 0.072，Cohen's d = 0.39（稳定性无显著变化）
- **τ 差异**：t(23) = 3.12，p = 0.008，Cohen's d = 0.64（长尾慢试次显著增多）

**解释**：反写 R 不仅平均反应时更长（μ 增大），而且更容易出现“卡顿”试次（τ 增大）。这可能意味着，在某些试次中，被试尝试直接旋转镜像字母失败，随后切换为“翻转→旋转”策略，导致反应时分布出现更长的指数尾巴。这一发现是单纯比较均值无法揭示的。

表 4 为各条件平均正确率（%）。所有条件正确率均在 96%–98% 之间。

| 旋转角度 | 正写 R（%） | 反写 R（%） |
| :------: | :---------: | :---------: |
|    0°    |    98.2     |    97.4     |
|   60°    |    97.8     |    97.0     |
|   120°   |    97.4     |    96.6     |
|   180°   |    96.8     |    96.2     |
|   240°   |    97.1     |    96.4     |
|   300°   |    97.7     |    96.9     |

重复测量 ANOVA 显示：旋转角度主效应不显著（F(5,115)=1.83, p=0.11）、字符类型主效应不显著（F(1,23)=2.15, p=0.16）、交互效应不显著（F(5,115)=0.92, p=0.47）。**支持 H3**——不存在速度‑准确性权衡。

### 4.6 个体差异：线性混合模型（LMM）结果

为检验心理旋转速度是否存在显著的个体差异（H4），我们拟合了线性混合模型 `RT ~ angle * type + (1 + angle | subject)`。固定效应估计见表 5，随机效应方差分量见表 6。

**固定效应**：

| 预测变量         | 估计值 (ms) | 标准误 | t 值 | p      |
| ---------------- | ----------- | ------ | ---- | ------ |
| 截距             | 546.2       | 12.4   | 44.1 | <0.001 |
| 角度（每10°）    | 27.5        | 1.8    | 15.3 | <0.001 |
| 字符类型（反写） | 101.5       | 8.2    | 12.4 | <0.001 |
| 角度 × 类型      | -0.3        | 0.5    | -0.6 | 0.55   |

**随机效应**：

| 随机效应     | 方差分量 | 标准差 |
| ------------ | -------- | ------ |
| 被试截距     | 185.6    | 13.6   |
| 被试角度斜率 | 0.012    | 0.11   |
| 残差         | 214.3    | 14.6   |

**关键发现**：随机斜率方差分量（被试角度斜率）通过似然比检验显著（比较有/无随机斜率的模型：χ²(1)=18.7, p<0.001），说明不同被试的心理旋转速度存在显著差异。边际 R²（仅固定效应）= 0.68，条件 R²（固定+随机效应）= 0.84，加入个体差异后模型解释力大幅提升。

**心理旋转速度的个体差异**：对每名被试单独计算旋转速度（0°–180° 回归），范围从 267°/s 到 489°/s（M = 364°/s, SD = 58°/s）。最慢与最快被试的速度差异达 222°/s，约为平均速度的 61%。这一发现印证了 LMM 的随机斜率结果。

### 4.7 心理旋转速度估计

基于 0°–180° 的线性回归，总体旋转速度为：

**速度 = 1 / 斜率 = 1 / 0.00275 s/° = 363.6°/s**

与 Cooper & Shepard（1973）报告的约 360°/s 高度一致。


## 五、讨论

### 5.1 核心发现与理论意义

本实验在严格复现 Cooper & Shepard（1973）经典结果的基础上，通过多种高级统计方法深化了对心理旋转机制的理解。

**线性性与对称性**：线性模型显著优于二次模型（ΔAIC = 5.6）且对称性检验不显著，为心理旋转的类比性和经济性提供了更强证据。若心理旋转是离散跳跃的，反应时应呈现阶梯函数而非光滑直线；若被试不选择最短路径，则 60° 与 300° 的反应时应显著不同。本实验的结果与 Shepard（1975）提出的“二级同构”概念完全一致——表象与外部客体在功能关系上同构，旋转角度与时间呈比例关系。

**镜像判断的额外成本**：反写 R 反应时平均长 102 ms，且 ex‑Gaussian 拟合显示该差异主要源于 μ 和 τ 的双重变化。μ 增大反映核心旋转效率降低（可能需要先翻转再旋转），τ 增大反映“卡顿”试次增多。这一发现超越了经典的“平均反应时差异”论述，揭示了镜像加工具有更高的变异性。可能的认知解释：某些试次中，被试可能尝试直接旋转镜像字母，失败后才切换为“翻转→旋转”策略，导致长尾分布。

**个体差异的显著性**：线性混合模型揭示个体旋转速度差异巨大（267–489°/s），且条件 R²（0.84）远高于边际 R²（0.68），说明个体差异在心理旋转任务中扮演重要角色。这一发现对认知能力的个体差异研究具有启示意义：心理旋转速度可能与空间智力、工作记忆容量等个体特质相关，未来研究可系统探索这些关联。

**与经典研究的一致性**：本实验的旋转速度（364°/s）、正/反写差异（~100 ms）、正确率（>96%）等关键指标均与 Cooper & Shepard（1973）原始报告高度吻合，验证了心理旋转效应的跨时间、跨文化、跨实验平台的稳健性。

### 5.2 对命题表征说的回应

本实验结果为命题表征说提出了三个直接挑战：

1. **线性性 vs. 离散性**：若命题转换是离散的（如每次改变一个特征），则反应时应在某些角度变化处出现“平台期”或阶梯函数。本实验的 R²=0.972 且二次项不显著，支持连续函数而非离散阶梯。
2. **180° 的最长反应时**：命题说预测 180° 应最容易（只需交换“上/下”和“左/右”），但本实验发现 180° 反应时最长。该差异无法通过命题框架调和。
3. **旋转方向的对称性**：命题表征通常具有方向性（如“左”和“右”不等价），难以解释 60° 与 300° 反应时完全相等。而表象的模拟表征可以自然地解释为：被试将表象沿最短弧旋转，方向由角度大小决定。

### 5.3 实验严谨性评估

**内部效度**：被试内设计消除了个体差异的混杂；完全随机化试次顺序避免了顺序效应和疲劳效应；所有控制变量保持恒定，确保了自变量是导致因变量变化的唯一原因。

**统计效度**：样本量 24 人在认知心理学中被试内设计中有足够的统计效力检测大效应量（偏 η²=0.85~0.79）。采用多种互补的统计方法（ANOVA、LMM、ex‑Gaussian）增强了结论的稳健性。

**构念效度**：字母 R 是不对称、高度熟悉的刺激，且被试未接受过反写 R 的训练，确保了任务必须依赖心理旋转而非语义记忆或特征匹配。经典的 Cooper & Shepard 范式已被广泛验证。

**外部效度**：被试均为大学生（18–24 岁），结论向其他年龄群体的推广需谨慎。此外，字母旋转任务与真实世界中的空间认知（如地图定向、物体识别）存在差异，生态效度有限。

### 5.4 批判性反思与替代解释

尽管本实验支持表象的模拟表征观点，但仍有若干值得批判性审视的方面：

**1. 心理旋转的连续性是否被高估？**  
本实验的反应时线性关系仅与连续性一致，但并非直接证据。Metzler（1973）的“不同时呈现”范式才是检验连续性的黄金标准。本实验未采用该范式，因此应谨慎表述：“线性函数与连续旋转假设相容，但不能排除离散阶梯函数在更精细角度间隔下出现的可能性。”未来研究应使用可变刺激间隔（SOA）直接验证旋转是否扫过中间角度。

**2. 镜像判断是否真正依赖旋转？**  
某些被试可能通过局部特征（如 R 的“腿”朝向）直接判断正反，而不进行完整旋转。本实验中，极快反应（200–300 ms）的试次虽仅占 1.2%，但可能存在。ex‑Gaussian 拟合中 τ 参数的增大也可能反映了策略切换（某些试次用特征匹配，某些试次用旋转）。未来研究可分析错误试次的模式或通过口头报告收集策略数据。

**3. 命题表征说的“弱版本”是否仍可解释？**  
虽然强命题说（离散符号操作）难以解释本实验结果，但混合模型（命题 + 模拟过程）仍有可能。例如，Anderson（1978）的 ACT* 理论允许命题网络在匹配过程中产生类似连续性的时间效应。因此，本实验不能完全“证伪”所有命题框架，而应视为对纯命题说的有力挑战，以及对表象独立性的支持。

**4. 生态效度与任务特殊性**  
“旋转字母判断正反”在现实生活几乎没有对应。专家（如飞行员、放射科医生）在熟悉刺激上的心理旋转速度显著快于新手，提示心理旋转具有高度的训练效应。本实验的被试均为普通大学生，结果可能低估了经过空间训练人群的旋转速度。未来研究可比较不同专业背景（如美术、工程、心理学）被试的旋转速度差异。

**5. 样本局限与统计效力**  
虽然 24 人在检测大效应时足够，但个体差异分析（如旋转速度与空间智力相关）需要更大样本。本研究发现的个体旋转速度差异范围达 222°/s，但受样本量限制，未能进一步探索与认知能力的关系。建议未来研究招募至少 60 名被试进行个体差异分析。

### 5.5 局限与未来方向

**本研究的局限**：

1. 刺激材料仅限字母 R，结论推广需谨慎
2. 未直接验证心理旋转的连续性（需 Metzler 范式）
3. 未记录策略报告，无法区分“旋转”与“特征匹配”策略的比例
4. 个体差异分析受样本量限制，未探讨与空间智力的关联

**未来研究方向**：

1. **连续性直接验证**：采用先后呈现两个图形的范式，操纵刺激间隔与旋转速度匹配，检验反应时是否恒定
2. **策略分离**：结合口头报告或反应时分布的双峰拟合，识别“旋转者”与“特征匹配者”的亚组
3. **个体差异研究**：招募大样本（N > 60），测量心理旋转速度与空间智力（如 mental rotation test）、工作记忆容量的相关
4. **神经机制**：结合 EEG 测量心理旋转过程中的 alpha 波段去同步化（反映顶叶激活），或 fMRI 定位顶内沟的活动
5. **应用研究**：将心理旋转范式应用于飞行员、外科医生等高空间要求职业的选拔与培训

### 5.6 结论

本实验成功复现了 Cooper 和 Shepard（1973）的经典字符旋转效应，并通过线性混合模型、ex‑Gaussian 拟合和线性/非线性模型比较等高级统计方法，进一步揭示了心理旋转的严格线性性、镜像判断的“卡顿”效应以及巨大的个体差异。主要结论如下：

1. 反应时与旋转角度呈严格线性关系，二次模型无显著增益；曲线关于 180° 完全对称，证明被试自动选择最短旋转路径。
2. 反写 R 的反应时平均长 102 ms，且 ex‑Gaussian 拟合显示该差异源于核心加工速度降低（μ 增大）和长尾慢试次增多（τ 增大）。
3. 个体心理旋转速度差异巨大（267–489°/s），线性混合模型显示随机斜率方差分量显著，提示个体差异是重要变异来源。
4. 所有条件正确率稳定 >96%，无速度‑准确性权衡。
5. 上述结果为双重编码理论提供了有力支持，并对命题表征说提出了直接挑战。
